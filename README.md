What is this project?

This project is a To-Do List Manager made using Python basics.
It helps the user to manage daily tasks by providing options to:

Add a task

View tasks

Remove a task

Exit the program


It is a menu-driven console application and is useful for beginners to understand how Python works in real-life mini projects.
Purpose of the project

The purpose of this project is to create a simple application that can organize daily tasks in an easy way.
It is designed for beginners to learn how to use:

Lists

Functions

Loops

Conditional statements

User input


This project shows how Python basics can be used to build a practical real-world application.


---

How the project works

When the program starts, it shows a menu with options:

1. Add Task


2. View Tasks


3. Remove Task


4. Exit



The user enters a choice.

Option 1 – Add Task

The user types a task name.

That task is stored in a Python list.

A success message is displayed.


Option 2 – View Tasks

The program checks whether the list is empty.

If there are tasks, it shows all tasks with numbering.

If there are no tasks, it shows a message saying no tasks are available.


Option 3 – Remove Task

First, the program displays the current tasks.

The user enters the task number to delete.

The selected task is removed from the list.


Option 4 – Exit

The program ends.

A while True loop is used to keep showing the menu until the user chooses Exit.

4. Conditional Statements

if-elif-else is used to check the user’s choice and perform the correct action.

Example:

If choice is "3" → remove task
